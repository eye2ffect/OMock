# 오목 AI 프로젝트

사용자와 대결할 수 있는 오목 인공지능(AI) 구현 프로젝트입니다. 

### 핵심 적용 알고리즘: Minimax (미니맥스)
기본적인 오목 게임에 AI가 스스로 판단하여 최적의 수를 찾을 수 있도록 **Minimax 알고리즘**을 핵심 로직으로 추가했습니다.

* **도입 목적:** 플레이어의 다음 수를 예측하여 승률이 가장 높은 자리를 찾기 위함입니다.
* **동작 방식:** 내가 돌을 두었을 때 상대방도 최선의 수를 둘 것이라고 가정하고, 여러 경우의 수를 계산하여 가장 유리한 좌표를 선택합니다.
* **구현 결과:** 2차원 배열로 보드 데이터를 관리하며, AI가 상대의 공격을 방어하거나 자신의 5목을 완성하는 최적의 수에 자동으로 착수합니다.
